# RPスタンドアロンアプリ — ドキュメント

## ファイル構成

| ファイル | 内容 | 読むタイミング |
|---------|------|---------------|
| [`RPスタンドアロンアプリ_設計書.md`](RPスタンドアロンアプリ_設計書.md) | **現行仕様（正本）** — アーキテクチャ・機能仕様・API・セキュリティ設計 | 最初に読む。実装時の参照 |
| [`plugin_development.md`](plugin_development.md) | プラグイン開発ガイド — hook・UI・action・テスト・コピー用雛形 | プラグインを新規作成・変更するとき |
| [`CHANGELOG.md`](CHANGELOG.md) | バージョン変更履歴 | 特定バージョンの変更内容を調べるとき |
| [`backlog.md`](backlog.md) | 未実装・保留・将来タスク | 次に何を実装するか検討するとき |
| [`model_list.txt`](model_list.txt) | 利用可能なモデル一覧 | モデル追加・確認時 |

## アーカイブ

旧統合設計書（v3.0〜v3.11）は [`archive/`](archive/) に読み取り専用で保管されています。

## 更新ルール

- **設計書**: コード変更を伴う機能追加・変更時に更新。現行仕様のみ記載し、過去の経緯は書かない
- **CHANGELOG**: バージョンリリース時に追記。変更内容・理由・影響ファイル・テスト件数を記録
- **backlog**: 新たな未実装項目や保留判断が発生したときに追記。実装完了時に削除
